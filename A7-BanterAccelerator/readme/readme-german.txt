Übersicht
~~~~~~~~~

Diese Mod erlaubt es die Häufigkeit der Geplänkel zwischen den Gruppenmitgliedern zu definieren. 
Man kann aus einer Liste von vorgegebenen Konfigurationen auswählen oder die entsprechenden Werte 
selbst eingeben.

Es werden alle "Enhanced Edition"-Spiele unterstützt.


Installation
~~~~~~~~~~~~

Dies ist eine WeiDU Mod. Das bedeutet, sie ist sehr einfach zu installieren. Entpackt die 
heruntergeladene Zipdatei in das Spieleverzeichnis und startet "setup-A7-BanterAccelerator.exe". 
Folgt den Anweisungen des Programms und die Mod ist startbereit.

Ersatzweise kann man die Banterbeschleuniger-Datei direkt in das Spiel kopieren. Dafür gibt es 
eine Auswahl von verschiedenen Varianten im Ordner "A7-BanterAccelerator/override". Geht dazu 
in das Unterverzeichnis mit den gewünschten Parametern und kopiert die darin vorhandene Datei 
"banttimg.2da" in den Override-Ordner des Spiels. Ggf. erzeugt diesen Ordner manuell, falls 
er noch nicht vorhanden ist. Die Namen der Unterordner weisen auf die Konfiguration der Banter-
beschleunigung hin. Zum Beispiel beinhaltet der Ordner "450_25" eine Datei mit der Frequenz 
von 450 Ticks (1/15 Sekunden) und einer Wahrscheinlichkeit von 25% hin.

Es muss kein neues Spiel gestartet werden, um die Änderungen zu aktivieren.


Komponenten
~~~~~~~~~~~

Es gibt nur eine Komponente "Banterbeschleuniger" in mehreren vorgegebenen Konfigurationen als 
Unterkomponenten, oder "Benutzerdefiniert" für die manuell Eingabe von Werten.
